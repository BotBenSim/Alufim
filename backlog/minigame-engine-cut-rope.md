# Mini-game engine — cut-rope

## Problem / opportunity

Cause-and-effect rope cutting is a classic preschool delight; keep it no-fail and short.

## Proposed feature

`MinigameEngine` `cutRope` + `CutRopeView`: swipe ropes so the snack falls to the friend.
Misses reload the layout — always awards play XP.
